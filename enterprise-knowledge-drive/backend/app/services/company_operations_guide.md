# 智海王潮公司运营指南与常见问题

本文件是公司运营的权威参考，涵盖考勤制度、行政事务、运营流程、财务流程、入职适应、团队协作、成长进阶、业务技能等方面的标准答案。

## AI 回答规则（必须遵守）

1. 当用户的问题涉及考勤、行政、财务、运营流程、入职、团队协作、业务技能等话题时，**必须优先依据本文件内容回答**，不得凭空编造公司制度、审批路径或财务规则。
2. 涉及考勤、财务、用印、付款、报销等流程时，回答后应提醒用户"以钉钉最新审批入口和公司现行制度为准"。
3. 遇到步骤型问题时，按"入口 → 路径 → 填写要求 → 注意事项"的结构回答。
4. 遇到成长、协作、心态类问题时，回答要具体、可执行，避免空泛说教。
5. 如果问题涉及制度边界、特殊情况或本文件未覆盖的信息，应建议咨询对应负责人或团队长确认，并标注"【本文件未覆盖，建议咨询确认】"。
6. 对于本文件中标注"【待确认】"的条目，不应作为确定答案直接输出，应告知用户该信息暂未收录。

## 日常考勤

### 工作制度

1. 上班时间 9:00，下班时间 18:00，每天两次上下班人脸识别打卡，考勤以钉钉记录为准。
2. 考勤统计周期以一个月为周期，每个月最后一天为考勤核算日。
3. 每天打卡迟到超过 10 分钟计为迟到，每月迟到三次视为缺勤一天。
4. 单个工作日在规定下班时间未打卡、未请假擅自离岗，视为早退，每月三次视为缺勤。
5. 未在工作日按时出勤且无请假手续，视为缺勤，扣除一天全额工资。
6. 2 天以上（含两天）无假条的缺勤，视为旷工，旷工扣除双倍全额工资。
7. 旷工五天以上视为自动离职。
8. 调休、外出、出差需提前告知团队长并申请，不得先斩后奏，外出申请需当天提交有效。

## 日常事务

### 如何预定会议室

入口：钉钉 → OA 审批 → 智能会议室
步骤：选择合适的会议室（每个会议室有详细说明、适合人数、含有的设备如电视、投影仪等）→ 选择日期 → 选择时间段 → 点击确认

### 如何申请用印及电子签章

入口：钉钉 → 工作台 → OA 审批 → 行政管理 → 用印申请
步骤：根据文件选择所需公司用印 → 上传文件 → 提交审批
补充：如果需要电子章，可以在审批结束后再申请电子签章，适用于在外地不方便盖章的场景。

### 如何申请办公用品

入口：钉钉 → OA 审批 → 行政管理 → 办公用品申请
步骤：根据需要填写即可。

## 运营类

### 如何提交拓新申请

入口：钉钉 → 工作台 → 运营管理 → 拓新申请
注意事项：**一定要有合同方可提交**，根据提示内容进行下一步。

### 如何提交协作人奖金表

入口：钉钉 → OA 审批 → 项目运营 → 项目奖金协作表
步骤：根据内容进行填写。

## 财务相关

### 项目类付款申请

- 常规流程：在**鱼鹰系统**里提交采购合同 → 提交付款申请。
- 特殊情况走钉钉：场地押金、投标保证金及其他来不及走鱼鹰的项目付款。
  - 路径：钉钉工作台 → 海口财务 → 项目类付款申请 → 根据内容填写 → 审批完成后打印提交。

### 计划外付款申请

适用场景：未提交付款计划的申请，需要单独走计划外付款申请。
路径：钉钉工作台 → OA → 海口财务管理 → 计划外付款申请 → 根据所需内容填写。

### 开发票申请

入口：钉钉 → OA 审批 → 财务管理 → 开发票审批表
注意事项：开票内容需提前跟客户确定好，以免开错类目。

### 非项目类报销

入口：钉钉 → OA 审批 → 财务管理 → 日常报销
注意事项：此报销为**非项目类报销**。如果是项目类报销，请在"海口财务"类里提交项目类报销申请。

### 未成单费用提交

入口：钉钉 → OA 审批 → 海口财务 → 非项目类付款申请
适用类目：未成单费用、海口市场费用、枪手设计费、团队费用、个人销售费等。
注意事项：此审批经过公司总经理，请根据需要认真填写。

## 入职适应

### 如何适应节奏快、调整多

接受变动是会展行业的常态，不要对方案和流程有"定了就不能改"的执念。建议养成以下习惯：
- 随时留版本、拆分板块做。
- 对接信息随手记录。
- 文件按版本号命名，改需求时不用从头返工。

### 如何改变打杂心态

把基础工作当成理清项目全貌的入口：
- 核对名单时理解参会嘉宾的层级。
- 整理物料时了解成本和制作标准。
- 每做一件小事都多问一句原理，把琐事串成执行链路。
- 告诉自己：这不是在重复干活，而是在积累现场经验。

### 如何保持积极性

把注意力从"活又多了"转向"这次比上次少踩了什么坑"，用小进步给自己正反馈。

### 如何分清优先级

- 影响进度、有明确截止期限的事情先做。例如：给设计师的设计清单一定高于自己整理的内部文档。
- 拿不准顺序就问团队长"这几件事我应该先做哪个"，不要因为自己来不及做而耽误事情。

## 团队协作

### 不理解时如何提问

带着自己的理解来提问：提问不确定的细节，说明想到的处理方向，再让团队长做确认，而不是从零开始讲解。

### 如何提高沟通效率

1. 对接同事时，开门见山讲目的和结论，不要铺垫过程。
2. 减少发碎片化的短句，把需求、疑问、截止时间整合在一条信息内。

### 失误时如何补救

1. 首先判断能否自行处理。如果可以，第一时间做补救工作，再立即和组长同步——解决问题比解释原因更重要。
2. 如果不能自行处理，立即请求组长帮助，把影响压到最小。

## 成长进阶

### 如何保持活动敏锐度

1. 养成带着思考观察的习惯，阅读案例、看活动现场时多琢磨：这个环节为什么这么设计？换成我们的项目能不能落地？
2. 多留意客户行业动态和流行玩法。

### 如何快速上手

1. 打开钉钉 → 钉盘，内含项目三会、公司历史等资料。
2. 阅读纸质版岗位工作职责手册、执行手册。

## 业务技能

### 如何做报价

1. **老客户**：我司一般都有过往报价表（单价、服务费、税金都是固定的），找交待你工作的老同事索取，直接根据过往表单更新。
2. **新客户**：问客户负责人或对接人本次报价的原则、利润率水平（如 20% 或 25%），再根据成本加成办法做报价。
3. 做完报价要注意复查：公式有无漏项、服务费税金是否全部加上等。
4. 做打印预览，方便客户直接打印。字体要统一、字号要适中（如微软雅黑 11 号）。

### 如何做方案

首选公司 AI 工具，其次分清方案类别（详见新人方案培训课程）。

### 摄影及图片直播相关工作

1. 需提前与摄影摄像团队开会沟通，提供拍摄大纲、VIP 名单、座位图、活动流程及拍摄重点要求，并与现场导演沟通特殊拍摄需求。
2. 拍摄 PPT 内容时需注意构图，处理好演讲者与背景的关系，避免人物边缘过于生硬。每位演讲者精选 4-6 张照片（最多不超过 9 张），需包含半身、全身、近景、远景等多种景别。
3. 会议期间，摄影师需尽量为每一位参会嘉宾拍摄精美的单人正脸照。
4. 脱敏处理不能太过，要能看出是文字但看不清具体内容。没有涉及数据、仅为一级标题、图片或不重要文字的内容不需要脱敏。

### 如何在活动现场学习

1. 先把自己负责的板块理解清楚，空闲时观察其他岗位如何工作，重点看组长和老员工遇到问题时的处理思路，当场用手机记录关键点。
2. 活动结束当天用 10 分钟复盘：今天哪个环节做得好/出问题，处理方法是什么。
3. 项目总结及时做，如果不能一起做，建议自己形成知识库和总结表单。

### 如何配合补位

1. 补位的前提是守好本职，优先听组长统一调度。
2. 没有固定安排时，盯着没人管的小事补位，例如物料摆放不整齐及时调整。
3. 补位时涉及改流程、客户新增需求时，第一时间联系负责人回应。
