# 开发说明

## 开发前端

前端开发使用 `Vite`，直接启动即可热重载：

```bash
cd /home/HAIKB/enterprise-knowledge-drive/frontend
npm run dev
```

默认访问地址：

- 本机：`http://127.0.0.1:5173`
- 内网：`http://你的内网IP:5173`

## 后端服务

后端使用 `systemd` 管理，开发时直接复用生产后端服务：

```bash
sudo systemctl status haikb-backend.service
sudo systemctl restart haikb-backend.service
```

健康检查：

```bash
curl http://127.0.0.1:9090/health
```

## 查看日志

实时查看后端日志：

```bash
journalctl -u haikb-backend.service -f
```

查看最近 100 行后端日志：

```bash
journalctl -u haikb-backend.service -n 100 --no-pager
```

实时查看 Nginx 日志：

```bash
journalctl -u nginx -f
```

## 生产发布

重新构建前端并重启生产环境：

```bash
cd /home/HAIKB/enterprise-knowledge-drive
sudo ./restart_prod.sh
```

首次启动或完整启动生产环境：

```bash
cd /home/HAIKB/enterprise-knowledge-drive
sudo ./start_prod.sh
```

停止生产后端：

```bash
cd /home/HAIKB/enterprise-knowledge-drive
sudo ./stop_prod.sh
```

查看生产状态：

```bash
cd /home/HAIKB/enterprise-knowledge-drive
./status_prod.sh
```

## 当前建议流程

1. 改前端代码：`cd frontend && npm run dev`
2. 改后端代码：`sudo systemctl restart haikb-backend.service`
3. 发布生产：`sudo ./restart_prod.sh`
